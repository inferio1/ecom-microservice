This is a work-in-progress e-commerce system built using Java Spring Boot and microservices architecture.

Project Status

Developed 3 microservices: Product, Order, User.

 APIs tested using Postman.
 
 Docker used for PostgreSQL & pgAdmin.

 Project is still under development; more features will be added.

Tech Stack

Java 17, Spring Boot 3

PostgreSQL, Docker

Postman for API testing

How to Run Locally

Clone the repository:

git clone https://github.com/your-username/ecom-microservices.git
cd ecom-microservices


Navigate to each microservice folder and run:

mvn spring-boot:run


Test APIs via the included Postman collection.
